#项目环境
Cocos2d-x版本:3.3 以上（至少3.x）   
开发环境：VS2013 + WIN10    
**注：由于生成的Android和Win8版本工程较大，没有上传。
#项目
###1.打开方式
找到proj.win32文件夹下，test.sln。然后使用VS2013打开即可，同时配置Cocos2d-x3.x版本，2.x是编译不通过的!    
###2.代码路径

###3.资源路径
